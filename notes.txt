-------------------

----------

True --> executes the block. False --> skip the block of code

elif block 
------------
syntax:
---------
elif condition:
    block of code

True --> executes the block. False --> skip the block of code

else block
----------
syntax:
----------

    block of code

True --> skip the block of code. False --> execute the code

Looping 
-----------
While --> Condition based loop.
For --> Sequence Based Loops.

>> Loops are used to execute ablock of code for multiple times..

While (No of iteration is not known)
---------
syntax:
----------
while condition:
    block of code

as long as the contition is True it will execute the block of code.

For Loop (iterating)(no of iteration is known)
----------
syntax:
----------

    block of code

Range  --> range sequence 
-------

range(start, end) --> range(4,44) --> [4,5,6....43]
